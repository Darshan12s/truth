# Truth Project

A web application with interactive UI components and media playback functionality.

## Features

- **Interactive UI**: Dynamic truth-based content display
- **Media Playback**: Audio song playback capability
- **Responsive Design**: Works across different screen sizes
- **Modern Styling**: Clean, contemporary CSS design

## Files

- `truth.html` - Main HTML interface
- `truth.css` - Styling and layout
- `truth.js` - JavaScript functionality
- `dashboard.html` - Dashboard page
- `songs/love-song.mp3` - Audio file

## Getting Started

1. Clone the repository
2. Open `truth.html` in a web browser
3. Interact with the UI elements

## Deployment

This project is automatically deployed to Vercel.
